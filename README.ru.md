# FoxClean

FoxClean — это мощный, бесплатный и современный инструмент с открытым исходным кодом для очистки и оптимизации macOS. Он сочетает в себе красивое нативное приложение SwiftUI с общим ядром Swift и универсальным интерфейсом командной строки (CLI) `fox`. Создан для пользователей, которые хотят получить полный контроль над производительностью своего Mac без телеметрии, подписок или скрытых платежей.

## 🚀 Ключевые Особенности

### 🖥️ Нативное Приложение SwiftUI
FoxClean поставляется с современным, быстрым и нативным интерфейсом приложения для macOS на основе PureMac.
- **Сканер Приложений**: Быстрый поиск и управление установленными приложениями.
- **Сканер Мусора**: Обнаружение и очистка ненужных файлов, кэшей и логов для освобождения места.
- **Обнаружение Остатков**: Поиск файлов, оставшихся от ранее удаленных приложений.
- **Статус Системы и Анализатор Диска**: Мониторинг состояния вашего Mac и визуализация использования диска в реальном времени.

![Clean](screenshot/clean.png)
*Глубокое сканирование системы и очистка от мусора в действии.*

### 🛠️ Общее Ядро `FoxCleanCore`
В основе FoxClean лежит надежное общее ядро, которое обеспечивает согласованность между приложением и CLI.
- **Пробная Очистка (Dry-run)**: Узнайте, что будет удалено, прежде чем вносить какие-либо изменения.
- **Удаление в Корзину**: По умолчанию безопасно перемещает файлы в Корзину, предотвращая случайную потерю данных.
- **Журналы Операций и Откат**: Ведет журналы операций в формате JSONL, позволяя вам отменить изменения при необходимости (rollback).
- **Очистка Проектов и Инсталляторов**: Быстрое удаление тяжелых каталогов проектов (таких как `node_modules` или `.build`) и оставшихся установочных файлов.

![Uninstall](screenshot/uninstall.png)
*Безопасное и полное удаление приложений и связанных с ними данных.*

### 💻 Мощный CLI `fox`
Для продвинутых пользователей и разработчиков CLI `fox` предлагает всесторонний контроль над вашей системой:
- Доступные команды: `scan`, `clean`, `uninstall`, `log`, `analyze`, `status`, `purge`, `installer`, `optimize`, `open`, `touchid` и `completion`.

![Options](screenshot/options.png)
*Дополнительные параметры и настройки для индивидуальной очистки.*

## 🔒 Безопасность Прежде Всего

Мы уделяем приоритетное внимание безопасности ваших данных.
- **Пробный запуск по умолчанию**: Все деструктивные действия CLI по умолчанию выполняются в режиме пробного запуска.
- **Перемещение в Корзину**: Используйте флаг `--confirm` для явного перемещения элементов в Корзину.
- **Безвозвратное Удаление**: Для настоящего безвозвратного удаления требуются флаги `--permanent` и `--confirm-permanent`, что предотвращает случайные катастрофы.

## 🏗️ Инструкции по Сборке

Чтобы собрать приложение FoxClean и CLI из исходного кода:

```sh
# Установите зависимости с помощью Homebrew
brew bundle

# Сгенерируйте проект Xcode
xcodegen generate

# Соберите приложение macOS
xcodebuild -scheme FoxCleanApp -destination 'platform=macOS' build

# Запустите тесты Swift
swift test

# Проверьте инструмент CLI
swift run fox --version
```

## 📜 Лицензия и Конфиденциальность

- **Никакой Телеметрии**: FoxClean не отслеживает вас, не собирает данные и не отправляет какую-либо информацию на удаленные серверы.
- **Никаких Подписок**: 100% бесплатно навсегда.
- **Лицензия MIT**: Открытый исходный код, созданный сообществом. Подробности смотрите в файле [LICENSE](LICENSE).
