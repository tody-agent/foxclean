# Delta: Clean UI

## ADDED Requirements

### Requirement: Clean Category Browser
FoxClean MUST expose cleanup categories with total size and count signals.

### Requirement: Multi-File Selection
The file browser SHOULD support multi-select cleanup decisions.

### Requirement: Size-Tiered Confirmation
Large cleanup actions MUST present stronger confirmation than small actions.

### Requirement: App List with Protection
System and protected apps MUST be visually locked and unavailable for uninstall.

### Requirement: Orphan Detection UI
The orphan view MUST show leftover files with best-guess ownership metadata.
