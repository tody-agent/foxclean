# Delta: Release Quality

## ADDED Requirements

### Requirement: Multi-Language Support
FoxClean MUST include localized resources with English fallback.

### Requirement: Accessibility
FoxClean SHOULD expose VoiceOver labels, keyboard navigation, contrast, and
reduced-motion support for public release.

### Requirement: Notarized Distribution
Public DMG distribution MUST be Developer ID signed, notarized, and stapled.

### Requirement: Homebrew Installation
FoxClean SHOULD provide Homebrew installation metadata for app and CLI.

### Requirement: Telemetry-Free
FoxClean MUST not include telemetry or analytics endpoints by default.
