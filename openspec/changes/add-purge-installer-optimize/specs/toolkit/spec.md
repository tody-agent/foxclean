# Delta: Toolkit

## ADDED Requirements

### Requirement: Project Artifact Detection
FoxClean MUST detect build artifacts only when a project marker is present.

### Requirement: Recent Project Safety
Recent files SHOULD be marked and left unchecked by default.

### Requirement: Custom Scan Paths
Project purge SHOULD support configurable scan roots.

### Requirement: Multi-Source Installer Discovery
Installer cleanup MUST label discovered installers by source.

### Requirement: Modular Optimization Tasks
Optimize MUST expose at least six independent tasks with toggles.
