# Delta: Analyzer and Monitor

## ADDED Requirements

### Requirement: Disk Tree Scan
FoxClean MUST scan an arbitrary directory and return a tree with byte sizes.

### Requirement: Treemap Data
FoxClean MUST provide treemap-ready weighted nodes for visualization.

### Requirement: Live System Metrics
FoxClean MUST provide system status data for CLI and GUI display.

### Requirement: Health Score
FoxClean MUST compute a 0-100 health score from multiple local factors.

### Requirement: Menu Bar Widget
FoxClean SHOULD expose menu bar quick actions for scan, open app, and quit.
