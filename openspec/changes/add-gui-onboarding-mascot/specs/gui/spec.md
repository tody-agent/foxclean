# Delta: GUI

## ADDED Requirements

### Requirement: First-Launch Onboarding
FoxClean MUST guide first-time users through welcome, Full Disk Access, and
ready states.

### Requirement: Sidebar Navigation
The main window MUST expose at least 10 navigation destinations with icons and
a routed detail view.

### Requirement: Dashboard Cards
The dashboard MUST show disk, scan, suggestion, and quick action surfaces.

### Requirement: Foxie Mood Reactivity
FoxieView MUST support idle, scanning, cleaning, success, error, sleeping,
curious, and dancing moods.

### Requirement: Theme and Localization
The GUI MUST support theme selection and localizable UI strings.
