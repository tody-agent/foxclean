# FoxClean

FoxClean एक शक्तिशाली, मुफ़्त और ओपन-सोर्स macOS क्लीनर और ऑप्टिमाइज़र है। यह एक सुंदर नेटिव SwiftUI एप्लिकेशन को एक साझा Swift कोर और बहुमुखी `fox` कमांड लाइन इंटरफ़ेस (CLI) के साथ जोड़ता है। उन उपयोगकर्ताओं के लिए डिज़ाइन किया गया है जो बिना किसी टेलीमेट्री, सब्सक्रिप्शन या छिपी हुई फीस के अपने Mac के प्रदर्शन पर पूर्ण नियंत्रण चाहते हैं।

![Home](screenshot/home.png)
*आपके Mac को स्वच्छ और अनुकूलित रखने के लिए एक आकर्षक, नेटिव SwiftUI इंटरफ़ेस।*

## 🚀 मुख्य विशेषताएँ

### 🖥️ नेटिव SwiftUI एप्लिकेशन
FoxClean, PureMac पर आधारित एक आधुनिक, तेज़ और नेटिव macOS एप्लिकेशन इंटरफ़ेस के साथ आता है।
- **ऐप स्कैनर**: इंस्टॉल किए गए एप्लिकेशन को तुरंत खोजें और प्रबंधित करें।
- **जंक स्कैनर**: स्पेस खाली करने के लिए अनावश्यक फ़ाइलों, कैश और लॉग की पहचान करें और उन्हें साफ़ करें।
- **ऑर्फ़न डिटेक्शन**: पहले अनइंस्टॉल किए गए एप्लिकेशन से बची हुई फ़ाइलों का पता लगाएँ।
- **सिस्टम स्टेटस और डिस्क एनालाइज़र**: अपने Mac के स्वास्थ्य की निगरानी करें और रीयल-टाइम में डिस्क उपयोग की कल्पना करें।

![Clean](screenshot/clean.png)
*डीप सिस्टम स्कैनिंग और जंक क्लीनिंग कार्य करते हुए।*

### 🛠️ साझा `FoxCleanCore`
इसके मूल में, FoxClean एक मजबूत साझा कोर का उपयोग करता है जो ऐप और CLI के बीच स्थिरता सुनिश्चित करता है।
- **ड्राई-रन क्लीनिंग**: कोई भी बदलाव करने से पहले देखें कि क्या हटाया जाएगा।
- **ट्रैश-फर्स्ट डिलीशन**: सुरक्षित रूप से फ़ाइलों को डिफ़ॉल्ट रूप से ट्रैश में ले जाएँ, आकस्मिक डेटा हानि को रोकें।
- **ऑपरेशन लॉग और रोलबैक**: JSONL ऑपरेशन लॉग बनाए रखता है, जिससे आप आवश्यकता पड़ने提示: परिवर्तनों को वापस ले (रोलबैक) सकते हैं।
- **प्रोजेक्ट पर्ज और इंस्टॉलर क्लीनअप**: भारी प्रोजेक्ट निर्देशिकाओं (जैसे `node_modules` या `.build`) और बचे हुए इंस्टॉलर्स को तुरंत हटा दें।

![Uninstall](screenshot/uninstall.png)
*एप्लिकेशन और उनके संबंधित डेटा को सुरक्षित रूप से पूरी तरह से अनइंस्टॉल करें।*

### 💻 शक्तिशाली `fox` CLI
पावर उपयोगकर्ताओं और डेवलपर्स के लिए, `fox` CLI आपके सिस्टम पर व्यापक नियंत्रण प्रदान करता है:
- उपलब्ध कमांड: `scan`, `clean`, `uninstall`, `log`, `analyze`, `status`, `purge`, `installer`, `optimize`, `open`, `touchid`, और `completion`।

![Options](screenshot/options.png)
*कस्टमाइज़्ड क्लीनिंग के लिए अतिरिक्त विकल्प और सेटिंग्स।*

## 🔒 सुरक्षा सर्वोपरि

हम आपके डेटा की सुरक्षा को प्राथमिकता देते हैं।
- **डिफ़ॉल्ट रूप से ड्राई-रन**: सभी विनाशकारी CLI कार्य डिफ़ॉल्ट रूप से ड्राई-रन मोड में होते हैं।
- **ट्रैश में ले जाएँ**: आइटम को स्पष्ट रूप से ट्रैश में ले जाने के लिए `--confirm` फ़्लैग का उपयोग करें।
- **स्थायी रूप से हटाना**: वास्तविक स्थायी हटाने के लिए `--permanent` और `--confirm-permanent` दोनों फ़्लैग की आवश्यकता होती है, जो आकस्मिक आपदाओं को रोकता है।

## 🗑️ FoxClean को अनइंस्टॉल करना

यदि आपको कभी भी अपने सिस्टम से FoxClean को हटाने की आवश्यकता होती है, तो आपके पास दो विकल्प हैं:

**1. मानक macOS तरीका:**
- `Finder` खोलें और `Applications` फ़ोल्डर में जाएँ।
- `FoxClean.app` को ट्रैश में खींचें (या राइट-क्लिक करें और "Move to Trash" चुनें)।
- ट्रैश खाली करें।

**2. CLI के माध्यम से पूर्ण निष्कासन:**
FoxClean और इसके सभी संबंधित डेटा (कैश, प्राथमिकताएं, लॉग आदि) को पूरी तरह से हटाने के लिए, आप ऐप को हटाने से पहले इसके स्वयं के CLI का उपयोग कर सकते हैं:
```sh
fox uninstall dev.foxclean.app --confirm
```

## 🏗️ निर्माण निर्देश

स्रोत से FoxClean ऐप और CLI बनाने के लिए:

```sh
# Homebrew का उपयोग करके निर्भरताएँ (dependencies) स्थापित करें
brew bundle

# Xcode प्रोजेक्ट जनरेट करें
xcodegen generate

# macOS एप्लिकेशन बनाएँ
xcodebuild -scheme FoxCleanApp -destination 'platform=macOS' build

# Swift परीक्षण चलाएँ
swift test

# CLI टूल सत्यापित करें
swift run fox --version
```
